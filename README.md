1. **Install dependencies** (from the project root):

   ```
   npm install
   ```

2. **Run the tests**:

   ```
   npx playwright test apps/saucedemo-project/tests/test.spec.ts
   ```

   Or run all tests in the project:

   ```
   npx playwright test
   ```

3. **View the HTML report**:

   ```
   npx playwright show-report
   ```

## Requirements

- Node.js >= 16
- Playwright
